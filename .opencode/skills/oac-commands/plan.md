# /oac:plan

1. **Phase 1: Analysis**
   - Use `explore` agent to find relevant files.
   - Read the architectural brief and PRDs.
2. **Phase 2: Discovery**
   - Identify dependencies and potential side effects.
   - Use `librarian` to check external docs or library versions.
3. **Phase 3: Plan Generation**
   - Create a `PLAT-###` story or task markdown file.
   - Breakdown into atomic sub-tasks.
   - Assign agents to each sub-task in the plan.
4. **Phase 4: Output**
   - Present the plan to the user for approval.
