# OpenCode Docs Skill

## Purpose

Guide agents to the official OpenCode documentation.

## When to Use

- Need authoritative guidance on OpenCode config/agents/plugins/tools.
- Need syntax for opencode.json, task permissions, models, themes, etc.
- Need instructions for agent creation or skill loading.

## Workflow

1. Navigate to <https://opencode.ai/docs/>.
2. Use sections:
   - Configure → Agents/Tools/Rules/MCP Servers
   - Usage → CLI/TUI/Web/IDE
   - Develop → SDK/Server/Plugins
3. Cite specific pages when summarizing.
