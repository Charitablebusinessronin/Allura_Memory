# Auto Skill Evolution

**Purpose:** Automatically detect patterns from session outcomes and evolve skills without human intervention.

**When to Use:**
- After epic-build-loop retrospective
- When pattern repetition detected (3+ times)
- For butter-smooth skill handoffs between sessions

**How It Works:**
1. Extract patterns from session events
2. Identify anti-patterns from failures  
3. Create/update skills automatically
4. Log evolution to Neo4j

---

## Pattern Detection

### From PostgreSQL Events

```typescript
// Query for repeated successful patterns
const patterns = await queryPostgres(`
  SELECT 
    context,
    COUNT(*) as frequency,
    AVG(EXTRACT(EPOCH FROM (ended_at - started_at))) as avg_duration
  FROM events
  WHERE event_type = 'skill_executed'
    AND status = 'success'
    AND timestamp > NOW() - INTERVAL '30 days'
  GROUP BY context
  HAVING COUNT(*) >= 3
  ORDER BY frequency DESC
`);
```

### Pattern Criteria

**Create New Skill When:**
- Pattern repeated 3+ times
- Success rate > 80%
- Clear trigger condition
- Documented approach

**Update Existing Skill When:**
- New edge case discovered
- Anti-pattern identified
- Better approach found
- User feedback received

## Skill Creation

### From Pattern

```typescript
async function createSkillFromPattern(pattern: Pattern) {
  const skill = {
    name: `auto-${slugify(pattern.name)}`,
    description: `Auto-generated from ${pattern.frequency} repetitions. ${pattern.description}`,
    trigger: pattern.trigger,
    action: pattern.action,
    examples: pattern.examples,
    source: {
      events: pattern.eventIds,
      sessionId: pattern.sessionId,
      createdAt: new Date()
    }
  };
  
  // Write skill file
  await writeSkillFile(skill);
  
  // Log to Neo4j
  await logToNeo4j('Skill', {
    name: skill.name,
    type: 'AutoGenerated',
    pattern: pattern.name,
    frequency: pattern.frequency
  });
  
  return skill;
}
```

## Skill Updates

### From Anti-Patterns

```typescript
async function updateSkillFromFailure(skillName: string, failure: Failure) {
  const skill = await loadSkill(skillName);
  
  // Add anti-pattern section
  skill.antiPatterns = skill.antiPatterns || [];
  skill.antiPatterns.push({
    pattern: failure.pattern,
    cause: failure.rootCause,
    fix: failure.resolution,
    date: new Date()
  });
  
  // Update guidance
  skill.warnings = skill.warnings || [];
  skill.warnings.push(`Avoid: ${failure.pattern}`);
  
  await writeSkillFile(skill);
  
  // Log to Neo4j
  await logToNeo4j('SkillUpdate', {
    skill: skillName,
    type: 'AntiPattern',
    pattern: failure.pattern
  });
}
```

### From Improvements

```typescript
async function updateSkillFromSuccess(skillName: string, improvement: Improvement) {
  const skill = await loadSkill(skillName);
  
  // Add better approach
  skill.patterns = skill.patterns || [];
  skill.patterns.push({
    name: improvement.name,
    approach: improvement.approach,
    benefits: improvement.benefits,
    date: new Date()
  });
  
  await writeSkillFile(skill);
  
  // Log to Neo4j
  await logToNeo4j('SkillUpdate', {
    skill: skillName,
    type: 'Improvement',
    pattern: improvement.name
  });
}
```

CREATE (s)-[:UPDATED_BY]->(u)
