1. **Do NOT modify existing Durham/soft-pop/brutalist/tangerine presets or components** unless absolutely necessary for integration. Create NEW allura-specific components.
2. **WCAG AA is mandatory.** Coral (#E85A3C) on white (#F5F5F5) fails contrast. Coral is ONLY for: deep-navy backgrounds, borders, or accents with sufficient surrounding contrast.
3. **Preserve all existing functionality.** The /memory page currently supports add, delete, restore, search, settings. All must continue working.
4. **TypeScript strict mode.** No `any` types. Use proper interfaces.
5. **Mobile-first.** All layouts must work on 375px width.
6. **Import the Outfit font** via next/font/google or CSS import.
7. **Do NOT break tests.** Baseline: 400 fail, 976 pass. Your changes must not add new failures.
8. **Use existing hooks.** `useMemoryList` is already implemented. Wire your UI to it.
