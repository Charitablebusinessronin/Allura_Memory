## Brand Foundation

### Archetypes
- **Caregiver (50%)** - Primary: nurturing, supporting, protecting community
- **Creator (30%)** - Secondary: crafting with intention, building with care
- **Explorer (20%)** - Tertiary: discovering new ways to connect

### Character
**Warm + Connected**

### Color Palette
| Color | Hex | RGB | Usage |
|-------|-----|-----|-------|
| Warm Yellow (Primary) | #FFC300 | 255, 195, 0 | CTAs, primary buttons, highlights |
| Deep Blue (Secondary) | #0581A7 | 5, 129, 167 | Secondary actions, links |
| Warm Green (Tertiary) | #BDBD0D | 189, 189, 13 | Success states, accents |
| Dark Gray (Text) | #142329 | 20, 35, 41 | Body text, headings |
| White/Cream | #F5F5F5 | 245, 245, 245 | Backgrounds |

### Typography
- **Primary:** Outfit (Google Fonts) - Headings, CTAs
- **Secondary:** Inter (Google Fonts) - Body text, UI elements

### Visual Language
- Droplet/arch shapes throughout
- Soft rounded corners (no sharp angles)
- 3-layer shadow system (ambient + directional + contact)
- Organic curves suggesting water ripples
- Clean, human-centered aesthetic
